Theory

The Equations

The Lorenz system is a three-dimensional autonomous ODE derived from a simplified convection model. It is deterministic, but for the classical parameter set it exhibits chaos because nearby initial states separate exponentially before the attractor's bounded geometry pulls them back into the same global structure.

dx / dt = sigma (y - x)
dy / dt = x (rho - z) - y
dz / dt = x y - beta z
Accuracy: the simulation uses a fixed-step classical fourth-order Runge-Kutta integrator instead of forward Euler, which sharply reduces visual drift for the same step size.
Dissipation: the vector field has constant divergence -(sigma + 1 + beta), so phase-space volumes contract over time and trajectories settle onto an attractor rather than filling all of 3D space.
Equilibria: the origin always exists, and when rho > 1 the nonzero equilibrium pair is located at (+/- sqrt(beta (rho - 1)), +/- sqrt(beta (rho - 1)), rho - 1).
