Method

What Is Being Simulated

The momentum field evolves through collision and streaming on a D2Q9 lattice, which is a standard lattice Boltzmann approximation for incompressible flow. The visible dye is not fed back into the fluid solve; it is advected separately as a tracer so you can see vortices, jets, and shear layers clearly in real time.

Momentum: BGK relaxation drives each cell toward its local equilibrium distribution after external forcing is applied.
Advection: dye uses a semi-Lagrangian backtrace, which is stable for interactive use and avoids exploding when the user injects strong impulses.
Rendering: the fluid image now blends low-resolution dye, curl-based shading, bloom, and highlight passes so the final result has more internal structure and depth.
