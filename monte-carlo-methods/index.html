Mode-Specific Parameters

The classic π experiment samples uniformly in the unit square and checks whether each point falls inside the quarter unit disk.

Estimator. If the hit probability is p, then π = 4p.
Variance source. Every point is a Bernoulli trial, so uncertainty shrinks like 1 / √n, not linearly.

Hit-or-miss integration estimates area by sampling inside a bounding rectangle and counting how many points fall below the chosen function.

All three examples live on [0, 1] and stay within a unit-height bounding box for clean comparison.

Method

Estimator Equations

Unbiasedness. Each estimator targets an expectation: hit probability, average function value, or discounted payoff. Sampling noise stays random, not systematically shifted.
Square-root law. Doubling accuracy typically requires about four times as many samples, which is why variance reduction is a core Monte Carlo design problem.
Reference solution. Each mode compares the random estimate against either a closed-form truth or a known analytic benchmark so the convergence story stays grounded.

Accuracy note: option pricing here assumes geometric Brownian motion and a frictionless Black-Scholes setting, so the Monte Carlo result is being compared against the matching model's exact price, not against real market behavior.
