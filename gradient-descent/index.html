About Gradient Descent

Gradient descent is an optimization algorithm used to minimize a function by iteratively moving in the direction of steepest descent as defined by the negative of the gradient. The algorithm follows the formula: xᵢ₊₁ = xᵢ - α▽fᵢ, where α is the learning rate and ▽fᵢ is the gradient at the current position.

This visualizer demonstrates how gradient descent works for different types of functions. Adjust the learning rate and starting point to see how they affect the algorithm's convergence.
